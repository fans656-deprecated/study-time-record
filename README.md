# 记录每天的学习时间

![Screenshot](https://raw.githubusercontent.com/fans656/image-hosting/master/20140818134314.png)

## 操作
空格键 开始/停止

## 显示
- 当前 session 的学习时间 (e.g. 01:23:04)
- 当天总学习时间 (e.g. 06:12:20)
- 当天剩余任务时间 (e.g. 01:47:60)

## 记录
所有历史学习时间，精确到 session

## 环境
- Python2.7.8
- PySide1.2.2
